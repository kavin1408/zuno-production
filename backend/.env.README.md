# Environment Configuration for Local Development

This `.env` file is configured for **local Docker development** using SQLite.

## Current Configuration

- **Database**: SQLite (local file: `zuno_v2.db`)
- **Authentication**: Supabase JWT Secret configured
- **AI Service**: OpenRouter API Key configured

## Switching Between Local and Production

### For Local Development (Current)
```bash
# DATABASE_URL is commented out - uses SQLite by default
# DATABASE_URL=postgresql://...
```

### For Production (Railway)
Uncomment the DATABASE_URL line:
```bash
DATABASE_URL=postgresql://postgres:x25CeFZHyZ9S6dE0@db.dcjsmglxckhllafocexe.supabase.co:5432/postgres
```

> **Important**: Never commit this `.env` file to Git! It contains secrets.
